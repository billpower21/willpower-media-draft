# Willpower Media Page — sandbox

This repo holds one page: `index.html`, the draft Media page for willpowerbrands.com.
It highlights panel recordings from Willpower events and the insights behind them.
See `README.md` for the full editing + publishing workflow.

## Voice — read this before writing ANY copy

Before drafting or editing any words on the page (session titles, takeaways, insight
breakdowns, pull-quotes, CTAs), **read `voice/about-bill.md` and follow it.** It is the
source of truth for Willpower's writing voice.

Quick rules (the file has the full version):
- Anchor every takeaway to a **named speaker + a specific number, tactic, or moment.**
  No vague generalities.
- **No em-dashes.** Use periods, commas, or parentheses.
- No "excited to announce," no manufactured urgency, and never the "Not X, it's Y" or
  "Not X. Not Y. The Z." patterns. State things positively.
- Sound like a sharp operator talking, not a press release. Keep it tight: a takeaway
  is one quotable line you could put on a slide.

## What you may edit

Only `index.html` (and content/voice files in this repo). This is a **sandbox**,
completely separate from the live site and safe to experiment in. Bill promotes the
page to the real willpowerbrands.com site himself once it's approved.
